# 18-share-article-twitter-medium

**As a** website visitor (logged in or not)  
**I want to** share an article on Twitter/X  
**So that** I can share content with my Twitter followers

**Acceptance Criteria:**
- Twitter share button is visible on article page
- Clicking button opens Twitter compose window
- Article title and URL are included in tweet
- User can customize tweet before posting
- Share count is tracked (optional)
- Button is mobile responsive
- **No login required** - anyone can share articles
