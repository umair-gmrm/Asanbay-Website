# 11-view-authors-section-high

**As a** website visitor (logged in or not)  
**I want to** browse the authors section  
**So that** I can discover authors and their articles

**Acceptance Criteria:**
- Authors section/page is accessible from main navigation
- Authors are displayed in a list/grid format
- Each author shows name, photo (if available), and brief bio/excerpt
- Authors can be sorted alphabetically or by number of articles
- Clicking on an author opens their profile page
- Page is mobile responsive
- **No login required** - anyone can view the authors section
