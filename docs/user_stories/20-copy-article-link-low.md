# 20-copy-article-link-low

**As a** website visitor (logged in or not)  
**I want to** copy the article link to clipboard  
**So that** I can share it through any platform or messaging app

**Acceptance Criteria:**
- Copy link button is visible on article page
- Clicking button copies article URL to clipboard
- User receives visual feedback (toast notification using HTMX)
- Copied link can be pasted anywhere
- Button is mobile responsive
- **No login required** - anyone can copy article links
