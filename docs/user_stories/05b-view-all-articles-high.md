# 05b-view-all-articles-high

**As a** website visitor (logged in or not)  
**I want to** see all published articles with category navigation  
**So that** I can browse articles by category and find content of interest

**Acceptance Criteria:**
- Articles page displays all published articles
- Articles are displayed in a list/grid format
- Each article shows title, **author (clickable, links to author profile)**, publication date, excerpt, and category
- Articles are paginated (e.g., 10 per page)
- **Categories are displayed in a secondary navigation panel** (sidebar or top navigation area)
- **All tags are listed in a side panel** (tag cloud or list format)
- Users can click on categories in the navigation panel to filter articles by category
- Users can click on tags in the side panel to filter articles by tags
- When a category or tag is selected, the article list updates to show filtered results
- Author names are clickable and link to author profile pages
- Articles are sorted by most recent first (default)
- Users can clear filters to see all articles again
- Page is mobile responsive (navigation panels adapt for mobile)
- **No login required** - anyone can view and filter on the articles page
- **Note:** Authors are independent entities managed by admin, not user accounts
