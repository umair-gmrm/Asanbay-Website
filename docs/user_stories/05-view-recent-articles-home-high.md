# 05-view-recent-articles-home-high

**As a** website visitor (logged in or not)  
**I want to** see recent articles on the home page  
**So that** I can discover the latest content from the society

**Acceptance Criteria:**
- Home page displays only recent articles (e.g., last 10-15 articles)
- Articles are displayed in a list/grid format
- Each article shows title, author, publication date, excerpt, and category
- Articles are sorted by most recent first
- Page is mobile responsive
- **No login required** - anyone can view the home page
- Link to "View All Articles" page is available
