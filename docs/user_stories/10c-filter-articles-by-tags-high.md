# 10c-filter-articles-by-tags-high

**As a** website visitor (logged in or not)  
**I want to** filter articles by clicking on tags  
**So that** I can discover articles with similar topics or themes

**Acceptance Criteria:**
- Tags are displayed on each article page (at the top)
- **All tags are listed in a side panel on the home page** (tag cloud or list format)
- Each tag is clickable and styled as a link/button
- **Clicking a tag from the home page side panel navigates to the articles page with that tag filter applied**
- Clicking a tag from an article page shows a list of all articles that have that tag (on articles page)
- Tag name is displayed prominently when filtering on articles page (e.g., "Articles tagged: 'Social Justice'")
- Filtered results show all articles with the selected tag
- Users can click on multiple tags to see articles with any of those tags (OR logic) or all tags (AND logic)
- Users can clear the filter to return to all articles
- Filtered articles maintain pagination
- Results are sorted by most recent first
- Page is mobile responsive
- **No login required** - anyone can filter by tags
- **Note:** Tags side panel on home page navigates to articles page with filter, filtering happens on articles page
