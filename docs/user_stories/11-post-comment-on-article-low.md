# 11-post-comment-on-article-low

**As a** logged-in user  
**I want to** post a comment on an article  
**So that** I can share my thoughts and participate in discussions

**Priority:** Low (Not required for initial release)

**Acceptance Criteria:**
- Comment form is visible **only** to logged-in users below article content
- For non-logged-in visitors, a "Login to comment" prompt is shown instead
- User can enter comment text
- Comment is posted and displayed immediately (using HTMX)
- Comment shows author name and timestamp
- **Login is required** to post comments (viewing comments does not require login)
- Comment form is mobile responsive
