# 12-view-author-profile-high

**As a** website visitor (logged in or not)  
**I want to** view an author's profile and their articles  
**So that** I can read all content from a specific author

**Acceptance Criteria:**
- Author profile page displays:
  - Author name
  - Author photo/avatar (if available)
  - Author bio/description
  - Contact information (if available)
- Author's articles are listed below the profile
- Articles are **ordered by recency** (most recent first)
- Each article shows title, publication date, excerpt, category, and tags
- Articles are paginated (e.g., 10 per page)
- Clicking on an article opens the full article page
- Author name on article pages links back to author profile
- Page is mobile responsive
- **No login required** - anyone can view author profiles
- **Authors are independent entities** - they don't need user accounts
