# 10b-filter-articles-by-category-high

**As a** website visitor (logged in or not)  
**I want to** filter articles by category  
**So that** I can find articles in specific topics or themes

**Acceptance Criteria:**
- Category navigation is available on the **home page (secondary navigation panel)** and articles page
- **Clicking a category on the home page navigates to the articles page with that category filter applied**
- Users can click on a category on the articles page to filter articles by that category
- Filtered results show only articles belonging to the selected category
- Category name is displayed prominently when filtering on articles page
- Users can clear the filter to see all articles again
- Filtered articles maintain pagination
- Results are sorted by most recent first
- Page is mobile responsive
- **No login required** - anyone can filter by category
- **Note:** Category navigation on home page navigates to articles page with filter, filtering happens on articles page
