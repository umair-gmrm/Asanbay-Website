# 02-register-new-user-high

**As a** new visitor  
**I want to** create an account  
**So that** I can participate in discussions and comment on articles

**Acceptance Criteria:**
- User can access registration page
- User can provide email, username, password, and confirm password
- System validates email format and password strength
- User receives confirmation email (optional)
- User is automatically logged in after registration
- Registration form is mobile responsive
