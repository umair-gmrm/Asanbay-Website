# 12-view-article-comments-low

**As a** website visitor (logged in or not)  
**I want to** see all comments on an article  
**So that** I can read discussions and other perspectives

**Priority:** Low (Not required for initial release)

**Acceptance Criteria:**
- Comments are displayed below the article
- Comments show author, timestamp, and content
- Comments are sorted by most recent or oldest (configurable)
- Comment count is displayed
- Comments are mobile responsive
- **No login required** - anyone can view comments
