# 01-login-user-high

**As a** website visitor  
**I want to** log in to my account (optional)  
**So that** I can post comments and access personalized features

**Acceptance Criteria:**
- User can enter email/username and password
- System validates credentials
- User is redirected to dashboard/homepage after successful login
- Error message displayed for invalid credentials
- Login form is mobile responsive
- **Note:** Login is optional - users can view articles and comments without logging in
