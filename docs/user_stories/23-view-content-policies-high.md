# 23-view-content-policies-high

**As a** website visitor (logged in or not)  
**I want to** view the Content Policies page  
**So that** I can understand the content policies and guidelines of the website

**Acceptance Criteria:**
- Content Policies page is accessible from main navigation/footer
- Page displays the title "Content Policies" (or configured title)
- Page displays the content configured by admin in Django admin
- Content is retrieved from database (not hardcoded)
- Page shows last updated date (if available)
- Page is mobile responsive
- Page follows the site's design theme (minimal, modern, elegant)
- **No login required** - anyone can view the Content Policies page
- Page is only visible if marked as active in admin
