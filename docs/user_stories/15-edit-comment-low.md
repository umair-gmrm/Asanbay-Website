# 15-edit-comment-low

**As a** logged-in user  
**I want to** edit my own comments  
**So that** I can correct mistakes or update my thoughts

**Priority:** Low (Not required for initial release)

**Acceptance Criteria:**
- Edit button is visible only on user's own comments
- Clicking edit allows inline editing
- Changes are saved and updated immediately (using HTMX)
- Edited comment shows "edited" indicator
- Original comment content is replaced with edited version
