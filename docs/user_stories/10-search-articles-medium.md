# 10-search-articles-medium

**As a** website visitor (logged in or not)  
**I want to** search for articles by keywords  
**So that** I can quickly find relevant content

**Acceptance Criteria:**
- Search bar is visible on article list page
- User can enter search keywords
- Results show matching articles
- Search works on title and content
- Results are displayed in same format as article list
- Search is mobile responsive
- **No login required** - anyone can search articles
